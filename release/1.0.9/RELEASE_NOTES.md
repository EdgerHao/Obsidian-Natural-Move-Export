# Natural Move/Export 1.0.9

## 新增

- 支持将系统剪贴板中的外部文件和文件夹粘贴到 Obsidian 文件夹。
- 支持 macOS Finder、Windows 文件资源管理器和 Linux 文件管理器。
- 在 Obsidian 文件夹右键菜单中新增“Natural move: 粘贴外部文件到此处”。
- 设置页新增右键菜单显示开关。
- 同名项目自动追加序号，避免覆盖已有文件。

## 使用方法

在系统文件管理器中复制文件或文件夹，然后右键单击 Obsidian 文件夹，选择“Natural move: 粘贴外部文件到此处”。

## BRAT Release 附件

请将 `main.js`、`manifest.json`、`styles.css` 作为三个独立附件上传。ZIP 文件可作为额外下载包，但不能替代这三个 BRAT 附件。
